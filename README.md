# College Enrollment System

This is a Laravel 8 based system running Tailwindcss, Alpinejs and Laravel Livewire. The system offers prospective students places to be students of the College. The prospective students must accept the offer and make a required payment, otherwise the offer is withdrawn. The Hod module enables Hods to enroll students belonging to courses offered in their department only whilst the admin module is more concerned with assigning of access control. The student is able to edit own profile and view payment details.

## Key Features
  - Authentication and Access Control System
  - Team Management
  - Image Handling System
  - Notification System
  - Payment System
  - Profile Management
